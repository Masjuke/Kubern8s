

### K8s Architecture


`Kubernetes stands with several components/item to run container orchestration` 

* Master
  - Controller Manager
  - Kubectl
  - Scheduler
  - Etcd
  - API
  
* Node
  - Kubelet
  - Proxy
  - Pods
